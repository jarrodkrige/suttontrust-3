# Chosen Style: **Bento Modular + Warm Minimalism**

### Why this style for first‑generation university students

*   ✅ **Low intimidation**: clear blocks instead of dense pages
*   ✅ **High scannability**: users may not know what they’re “supposed” to click yet
*   ✅ **Modular**: content can grow as support programs expand
*   ✅ **Emotionally safe**: warm, human, non‑corporate

This is the same family used by **Notion, Stripe, Vercel**, but *softened* for community and wellbeing.

***

## 1. Brand Tone & UX Principles (Foundation)

**Design values**

*   Supportive, not institutional
*   Clear, not clever
*   Empowering, not patronising

**UX rules**

*   Every screen answers: *“What can I do next?”*
*   No jargon without explanation
*   Navigation always visible
*   One primary action per section

***

## 2. Color Palette (Accessible & Warm)

### Core Colors

```txt
Ink (Primary Text):        #1E1E1E
Cloud (Background):       #FAFAF9
Sand (Card Background):   #F3F4F2
Leaf (Primary CTA):       #2F7D6D
Sky (Secondary Accent):   #4F8FEA
Sun (Highlight):          #F4C95D
```

### Usage

*   **Cloud** → full page background
*   **Sand** → bento cards
*   **Leaf** → main buttons, key links
*   **Sky** → secondary actions, links
*   **Sun** → encouragement highlights (badges, progress)

✅ All contrast pairs pass WCAG AA for body text.

***

## 3. Typography System

### Headings

**Font:** `Inter` or `Source Sans 3`

*   H1: 36px / Semibold
*   H2: 28px / Semibold
*   H3: 20px / Medium

### Body

*   Size: 16px
*   Line-height: 1.6
*   Weight: Regular

### Tone rule

*   Use **sentences**, not slogans
*   Prefer *“You can…”* over *“We provide…”*

***

## 4. Layout: Bento Grid System

### Base Grid

*   12‑column layout
*   Cards span 3–6 columns
*   16–24px internal padding
*   Rounded corners: **16px**

### Example Homepage Layout

    [ Welcome / Who this is for ]  [ Find Support ]

    [ Scholarships ] [ Mentors ] [ Housing Help ]

    [ Step-by-step: Your first semester ]

    [ Real Stories from Students ]

Each card = one idea, one action.

***


**Copy examples**

*   “Get help with applications”

***

### Secondary Button

```css
border: 1px solid #D1D5DB
background: #FFFFFF
color: #1E1E1E
```

***

### Card Component (Bento Unit)

```txt
Background: #F3F4F2
Padding: 20px
Radius: 16px
Shadow: none (use color separation only)
```

**Card Structure**

1.  Small icon (top-left)
2.  Clear title
3.  1–2 line explanation
4.  One action link

***

## 6. Icon Style

**Style**

*   Outline icons
*   Rounded strokes
*   Simple metaphors (no abstractions)

**Examples**

*   🎓 Graduation cap → academics
*   🧭 Compass → guidance
*   💬 Chat bubble → mentoring
*   🏠 House → housing
*   💼 Folder → paperwork

➡️ Use Lucide, Phosphor, or Heroicons (outline only).

## 8. Micro‑Interactions (Subtle but Important)

*   Buttons slightly darken on hover (no bounce)
*   Focus rings always visible
*   Cards lift **1px max** on hover
*   Progress indicators for multi‑step guides

No flashy animations. Calm = trust.

# 🎨 Visual Designer

You produce the imagery: hero, secondaries, portraits, dividers, alt
heroes — whichever the page wants. You'll mostly use **Microsoft Copilot's
image generator**.

Multiple designers in the team — you all contribute together. The agent
picks the strongest images and reasons about best placement (hero, voice
portraits, buddy photo, dividers, closing).

## What lives in this folder — pick whichever calls to you

- `hero.png` — the main hero image (or `alt-heroes/` for variants)
- `secondary-images/` — supporting images for content sections
- `icons/` — small graphics for feature lists
- `portraits/` — illustrations for voice quotes
- `buddy.png` — the friendly closing portrait
- `dividers/` — section dividers, decorative shapes
- `backgrounds/` — subtle background textures
- `closing-image.png` — a quiet, warm final image
- `palette.md` — colour suggestions for the team's CSS
- `style-notes.md` — what makes the imagery feel cohesive (line weight,
  palette, mood)

**Claim a lane with your teammates first** — hero, secondaries,
portraits/buddy, dividers/backgrounds, closing. Agree style and palette in
the first few minutes so the page reads as one piece.

## Coordinate with

- **Other Visual Designers** — agree style/mood early; share your palette
- **Software Developers** — they propagate your palette into the CSS
- **Copywriters** — your imagery should match the tone they're setting
- **Accessibility Champion(s)** — they'll write or check alt text

## First 2 minutes

1. Sign in to **copilot.microsoft.com** with your role-card account.
2. Click the **image** option (or say "make me an image of...").
3. Try a prompt below. Iterate. Save the winner with a clear filename.
4. Commit.

## Prompts that work

- *"Warm, hopeful illustration of three diverse UK sixth-form students looking at a university campus from a hill. Soft pastel colours, illustrative, not photorealistic. Editorial magazine style. Wide aspect ratio for a website hero."*
- *"Make it warmer."* / *"Make it less corporate."* / *"Less stock-photo, more illustration."*
- *"Same scene, different angle."* / *"Same style, different subject."*
- *"Suggest a 4-colour palette inspired by this image."*
- *"Critique this image idea: [paste]. Does it feel inviting to a 16-year-old?"*
